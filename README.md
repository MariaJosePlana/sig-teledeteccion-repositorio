# Portafolio SIG y Teledetección

Proyectos desarrollados en el marco del **Diplomado en Manejo de Información Espacial** de la Universidad Mayor (2023). Cada carpeta contiene su propio README, los productos cartográficos y la documentación del laboratorio correspondiente.

## Proyectos

| # | Proyecto | Software | Descripción |
|---|----------|----------|-------------|
| 01 | [Clasificación forestal — Parque Nacional Conguillío](01-clasificacion-forestal-conguillio) | ENVI | Clasificación de tipos de vegetación e índices espectrales (NDVI, EVI, ARVI, SIPI, NDRE) a partir de imágenes hiperespectrales Hyperion. |
| 02 | [Mapa y aplicación web de pobreza multidimensional](02-app-web-pobreza-multidimensional) | ArcGIS Online / Web AppBuilder | Visualización web interactiva de la pobreza multidimensional comunal de la Región Metropolitana (CASEN 2017). |
| 04 | [Riesgo de erosión post-incendio — Maule 2017](03-erosion_post-incendio) | ArcGIS | Índice multicriterio de riesgo de erosión en el área afectada por el Gran Incendio del Maule. |
| 05 | [Ordenamiento territorial — Alhué](05-ordenamiento-territorial-alhue) | ArcGIS | Análisis de ordenamiento territorial y zonificación de la comuna de Alhué. |

## Herramientas

ArcGIS y ArcGIS Online, ENVI, teledetección óptica e hiperespectral, análisis multicriterio y Web GIS.
